# Rook & Roll

This repository is now bootstrapped for **GitHub Pages** hosting so you can quickly evolve it into a web-playable game prototype.

## What's included

- `index.html` as a starting landing page.
- `styles.css` for basic styling.
- `.github/workflows/pages.yml` to deploy on every push to `main`.

## Enable GitHub Pages

1. Push this repo to GitHub.
2. In **Settings â†’ Pages**, set **Source** to **GitHub Actions**.
3. Push changes to `main` and GitHub will publish the site.
